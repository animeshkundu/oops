Rules catalog

oops ships with 175+ rules covering the commands developers use every day. Rules are compiled for speed and run lazily for performance.

Git & VCS

Push, branch, checkout, commit, stash, submodule, and more.

git_push git_checkout git_commit_amend git_branch

Package managers

APT, Brew, Cargo, npm, pip, conda, yarn, dnf, yum.

brew_install npm_run_script cargo_no_command pip_unknown_command

System commands

sudo, cd, chmod, mkdir, rm, cp, and related utilities.

sudo cd_mkdir chmod_x mkdir_p

Cloud & DevOps

AWS, Azure, Terraform, Docker, Kubernetes, Heroku.

aws_cli docker_login terraform_init kubectl_not_command

Languages & frameworks

Go, Java, Rails, React Native, Django, Maven.

go_run javac rails_migrations_pending react_native_command_unrecognized

Shell utilities

grep, sed, tar, find, ssh, and common CLI helpers.

grep_arguments_order tar_twice ssh_known_hosts history
