Install in one command

These scripts download the latest release binary and place it on your PATH. Choose the command for your platform and run it once.

macOS

Runs on Intel and Apple Silicon.

curl -fsSL https://raw.githubusercontent.com/animeshkundu/oops/master/scripts/install.sh | bash

Linux

Supports x86_64 and aarch64.

curl -fsSL https://raw.githubusercontent.com/animeshkundu/oops/master/scripts/install.sh | bash

Windows

PowerShell will install and update your PATH.

iwr -useb https://raw.githubusercontent.com/animeshkundu/oops/master/scripts/install.ps1 | iex

Prefer package managers?

Use your preferred ecosystem instead of the scripts.

  • Homebrew: brew install animeshkundu/tap/oops
  • Cargo: cargo install oops
  • Scoop: scoop install oops
  • Arch (AUR): yay -S oops

Shell integration

Add oops to your shell once so it can re-run fixes.

Bash / Zsh

eval "$(oops --alias)"

Fish

oops --alias | source

PowerShell

Invoke-Expression (oops --alias | Out-String)

Verify the install

Confirm the binary is available and shell alias loads.

oops --version
# then try a typo
ls -lahh
