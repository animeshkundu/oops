Quickstart

Follow these four steps to see your first oops correction. This is perfect for onboarding new teammates.

1

Install the binary

Use the one-line installer from the installation page.

2

Add the alias

eval "$(oops --alias)"
3

Trigger a typo

git statsu

Oops will catch the typo and suggest git status.

4

Run oops

oops

The suggestion list will appear instantly.

Pro tip: auto-run the top fix

Skip the selection UI by adding -y.

oops -y
