Migration guide

oops is designed for drop-in adoption. Keep your shell habits, reuse your env vars, and enjoy 10x faster corrections.

Why switch?

  • ~30ms startup vs ~300ms for Python.
  • Single Rust binary, no runtime dependency.
  • Compatible env vars and alias names.
  • Built-in parity checker for rule coverage.

Quick swap (2 minutes)

  1. Install oops using the one-liners.
  2. Replace thefuck alias with oops.
  3. Convert config from Python to TOML.

Shell alias mapping

Keep muscle memory by setting a compatible alias.

TF_ALIAS=fuck oops --alias

This makes fuck call oops instead.

Config migration

Oops uses TOML instead of Python, but env vars stay the same.

# Before (settings.py)
rules = ['ALL']
require_confirmation = True

# After (config.toml)
rules = ["ALL"]
require_confirmation = true

Parity & coverage

Check your missing rules compared to thefuck.

cargo run --bin check_parity
